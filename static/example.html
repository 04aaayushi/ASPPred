Steps To Follow

Step 1:   Enter the Peptide Sequence.

      Sample input sequence: RRLGVMRW

Step 2:   Click the predict button.


Result Interpretation

ASPPred server discriminates the antisepsis peptides from non-antisepsis peptides. The ASPPred server uses the Random Forest classifier for prediction.

Sequence Format:- Amino acid sequences should be entered in the one-letter code and the length of the peptide entered must be greater than 4. All the non standard characters like [*&^%$@#!()_+~=;'",<>?.\|} are considered as invalid sequence. The warning is displayed if the user submits unnatural amino acids or no sequence is submitted.
